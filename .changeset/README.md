# Changesets
このフォルダは `@changesets/cli` によって自動生成されました。
Changesets は、マルチパッケージ（モノレポ）やシングルパッケージのリポジトリにおいて、コードのバージョニング（バージョン管理）や公開作業を支援するためのビルドツールです。

詳細なドキュメントについては、[公式リポジトリ](https://github.com/changesets/changesets)をご覧ください。

また、プロジェクトをスムーズに開始するための[よくある質問 (FAQ)](https://github.com/changesets/changesets/blob/main/docs/common-questions.md)もドキュメント内に用意されています。
